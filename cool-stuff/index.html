DeetsJack

building

A blackjack trainer disguised as a blackjack game.

Play full hands, then flip the trainer card to see the live math — dealer bust odds, your bust odds, and the EV of every move. It scores decision quality, not luck: the +EV move is “right” even when the hand loses. The pure-Swift EV engine is unit-tested against published basic-strategy numbers.

Swift 6 iOS probability engine

Tools & systems

DeetsKit

active

The design system behind every Deets app.

A Swift package holding the shared visual language — 8 color themes and 4 skins (glass, comic, retro, sunshine), semantic typography, and reusable glass components — so every app looks and feels identical without copy-pasting a theme layer. Ships its own WCAG contrast auditor and an on-device AI palette generator.

Swift package iOS / macOS design system
GitHub ↗

DeetsOTD

DeetsRadio

live

Shared listening rooms, one Durable Object each.

The worker behind this site’s radio tab: hibernatable WebSockets carry a communal queue with synced playback, an alarm keeps empty rooms ticking, and a D1 registry matches songs across catalogs. The room owns a clock, never audio — zero dependencies, free-tier discipline throughout.

Cloudflare Workers Durable Objects WebSockets
GitHub ↗

DeetsLeague

live

The live stats engine behind the League tab.

A dependency-free worker that proxies the Riot API so the key never reaches the browser, normalizes matches into a D1 store, and runs a budgeted cron crawler that backfills each player’s history ~1,000 games deep without ever blowing the rate limit.

Cloudflare Workers D1 Riot API
