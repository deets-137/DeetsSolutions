Apps & experiences

DeetsMusic for Windows

active

A full Apple Music player for Windows 11.

Frameless custom chrome, full DRM playback with a live-synced gapless queue, Apple Music sign-in with a local SQLite library cache, and a deeply themeable UI — 6 themes × 5 skins driven by a token system.

Tauri v2 Rust TypeScript WebView2

DeetsLife

building

A walkable 3D museum of the music of my life.

An interactive art project organized by era — jukeboxes and vinyl records are the interaction hubs, with photos and media layering in over time. Ships as an interactive build plus a recorded cinematic walkthrough.

Unity / URP C# Blender

DeetsJack

concept

A blackjack trainer disguised as a blackjack game.

Play full hands, then flip the trainer card to see the live math — dealer bust odds, your bust odds, and the EV of every move. It scores decision quality, not luck: the +EV move is “right” even when the hand loses.

Swift 6 iOS probability engine

DeetsBoard

building

A spatial workspace for designing board games.

Design cards, units, pieces, tiles, and rules with friends on a 3×3 spatial canvas. Native macOS on Core Data + CloudKit; the successor to Henhouse, my earlier web take on the same idea.

Swift macOS CloudKit

DeetsTrips

building

Trip planning where friends follow along.

One organizer plans the trip; everyone else installs the app and rides shotgun. SwiftUI + SwiftData, with CloudKit sync on the roadmap.

SwiftUI SwiftData iOS

Rogers

building

A student-forward Learning Management System.

An LMS that brings color and joy to education software — modules under one umbrella so teachers get holistic views of students, counselors see the whole student journey, and schools set up robust systems in one easy go. Full-stack build with a front end, back end, and Docker deployment.

TypeScript full-stack Docker

DeetsHealth

building

Tools & systems

DeetsKit

active

The design system behind every Deets app.

A Swift package holding the shared visual language — 8 color themes and 4 skins (glass, comic, retro, sunshine), semantic typography, and reusable glass components — so every app looks and feels identical without copy-pasting a theme layer.

Swift package iOS / macOS design system

Harness

active

A local agent harness for Ollama-served models.

FastAPI + WebSocket streaming into a browser UI built as a self-arranging bento panel system that the model itself can rearrange, with tool packs for chess, D&D, blog editing, and coding — plus a Discord bridge.

Python FastAPI local LLMs

DeetsEdu

building

Modeling how US education is actually doing.

A data pipeline over the Stanford Education Data Archive and NCES data — district-level test scores as queryable timeseries, built with DuckDB and shipped as portable SQLite, headed toward public visualizations.

Python DuckDB ~1.4 GB of data

DeetsOTD

built

Every link ever posted in a Discord channel, exported.

Reads the hyperlinks out of a channel and writes them to deduped, grouped Markdown and CSV — as a slash-command bot, a CLI, or a local web trigger. It’s where this site’s Song of the Day journal comes from.

Python Discord bot
